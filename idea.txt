整个计算器分成 屏幕（screen） 和 按钮 (button)

按钮分为 数字（0-9）和 运算符（）
运算符分为 基本运算（加减乘除），小数点，等号，消除（AC）,平方（x2） 

用户点击按钮->获取按钮类型->同时将数字输出到屏幕->根据对应按钮进行运算-> 点击= 获得结果 并将结果输出

	

计算器初始屏幕显示为0
用户点击按钮  	
	- 为数字 屏幕显示对应数字 0 除外
	- 为运算符  当前屏幕上的数字 保存为第一个数字	   
	- 为=，x2，AC 是无反应	
	- 为 . 显示为 0. 一组数字最多只能有一个小数点


当用户按点 
0. 用户输入过点 后点        	// return
1. 用户什么都没输入 直接按点 	 // 0 + .	
2. 用户输入数字按点						// 数字 + .	
3. 用户输入operator 后按点		// 0 + .	

当用户按operator (+ - * /)
1. 用户什么都没按 直接按operator 			// 0 + 'operator'
2. 用户输入完数字 按operator 					//
3. 用户输入完operator 再按operator		//
4. 用户按完= 在按operator

当用户按= 
1. 用户什么都没按 直接按=
2. 用户输入完数字 按=  
3. 数字 + ‘operator’ 		//
4.
